# Observator — CSV to Database Mapping

## How Real Data Flows: CSV → PostgreSQL → API → Frontend

### Dimension Tables (Reference Data)

| DB Table | CSV Source | Key Columns | Rows |
|----------|-----------|-------------|------|
| `dim_occupation` | `4_taxonomy_esco/esco_occupations.csv` | esco_uri → code_esco, occupation_en → title_en, occupation_ar → title_ar, isco_group → code_isco | 3,043 |
| `dim_skill` | `4_taxonomy_esco/esco_skills.csv` | esco_uri → uri_esco, skill_en → label_en, skill_ar → label_ar, skill_type | 13,960 |
| `dim_institution` | `2_supply_education/uae_he_institutions_master.csv` | institution_en → name_en, institution_ar → name_ar, emirate_en → emirate | 151 |
| `dim_sector` | Pre-seeded (21 ISIC sectors) | label_en, code_isic | 21 |
| `dim_region` | Pre-seeded (7 emirates) | region_code, emirate | 7 |
| `dim_time` | Pre-seeded (2015-2035 daily) | date, year, month, quarter | ~7,300 |
| `dim_discipline` | Pre-seeded (12 disciplines) | label_en, code_isced | 12 |
| `crosswalk_soc_isco` | `7_crosswalks/bls_isco08_to_soc2010_crosswalk.csv` | soc2010_code → soc_code, isco08_code → isco_code | 1,126 |

### Fact Tables (Actual Data)

| DB Table | CSV Source | Key Mapping | Rows |
|----------|-----------|-------------|------|
| `fact_occupation_skills` | `4_taxonomy_esco/esco_occupation_skill_map.csv` | occupation_uri → occupation_id (via dim_occupation.code_esco), skill_uri → skill_id (via dim_skill.uri_esco), relation_type | ~126,000 |
| `fact_demand_vacancies_agg` | `3_demand_jobs/linkedin_uae_job_postings_2024_2025.csv` | date → time_id, location → region_code, occupation → occupation_id (ISCO major group), experience → experience_band, each row = demand_count=1 | 36,923 |
| `fact_supply_talent_agg` | `1_supply_workforce/fcsc_employment_master.csv` | year → time_id, pct → supply_count (scaled), gender, nationality, dimension_type/value → age_group/education_level | 1,394 |
| `fact_ai_exposure_occupation` | `6_ai_impact/aioe_occupation_scores.csv` | SOC Code → (crosswalk) → ISCO → occupation_id, AIOE → exposure_z | 774 |
| `fact_supply_graduates` | TODO: Bayanat education data | year, institution, discipline, gender, nationality, graduates_count | TBD |
| `fact_forecast` | Generated by forecasting engine | Not from CSV — computed at runtime | TBD |

### Materialized Views (API queries these)

| View | Built From | What Frontend Shows |
|------|-----------|-------------------|
| `vw_supply_talent` | fact_supply_talent_agg + dim_* | Supply-side charts |
| `vw_demand_jobs` | fact_demand_vacancies_agg + dim_* | Demand-side charts |
| `vw_gap_cube` | supply + demand joined | **Skill Gap Index (SGI)** — main dashboard |
| `vw_ai_impact` | fact_ai_exposure_occupation + dim_occupation | AI Impact page |
| `vw_supply_education` | fact_supply_graduates + dim_* | University page |
| `vw_forecast_demand` | fact_forecast + dim_* | Forecast page |
